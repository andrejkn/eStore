## eStore
Online store

## Getting Started

Use our [starter script](https://www.npmjs.com/package/rangle-starter), with
`react-redux-starter` as the `techStack` argument.

## npm scripts

### Dev
```bash
$ npm run dev
```

Open `http://localhost:3000` in your browser.

### Tests

#### Single Run
```bash
$ npm run test
```

#### Watch Files
```bash
$ npm run test:watch
```

#### Coverage
```bash
$ npm run cover
```

### Production
```bash
$ npm start
```
